### PARA RODAR O PROJETO LOCALMENTE:
- Abrir uma pasta e dar 'git clone' com o link do repositório
- Acessar a pasta 'db" e rodar o script SQL para a criação do banco localmente
- Alterar o arquivo '.env' na raiz do projeto com a senha do seu banco
- Abrir o Terminal na pasta do projeto e dar 'npm install', para instalar as dependências
- Ainda no Terminal, digite o comando "node server.js" para rodar o projeto
- Teste as funcionalidades do sistema e a conexão com o bando de dados, fazendo alterações como adicionar clientes, produtos e pedidos
